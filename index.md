プライバシーポリシー

本アプリ（以下、「本アプリ」といいます）は、ユーザーのプライバシーを尊重し、
個人情報の保護に努めています。本アプリが収集する情報、その利用目的、
第三者提供の有無、データの管理方法について以下に明示します。

1. 収集する情報
本アプリでは、以下の情報を収集する場合があります。

(1) 匿名ログイン情報（Firebase Authentication）
　本アプリでは、ユーザー識別のため Firebase の匿名ログイン機能を使用します。
　メールアドレスや氏名など、個人を特定できる情報は収集しません。

(2) スコアデータ（Firestore）
　ユーザーがプレイしたスコア、レベル、ニックネーム、国コードを保存します。
　これらはオンラインランキング表示のために使用されます。

(3) 端末の国コード
　端末の地域設定から国コード（例：JP, US）を取得します。
　位置情報（GPS）は一切使用しません。

(4) Game Center の利用
　本アプリでは、オンラインランキング機能のために Apple の Game Center を使用します。
　Game Center にログインしている場合、Apple が提供するプレイヤー識別子（Player ID）を利用してスコアを管理します。
　本アプリは Game Center のプレイヤー名やメールアドレスなど、
　個人を特定できる情報を取得しません。

(5) 広告ID（Google AdMob）
　広告配信のため、Google AdMob が広告IDを自動的に取得する場合があります。
　これらは Google のプライバシーポリシーに基づいて管理されます。

(6) ローカル保存データ
　オフラインランキングのため、端末内にスコアを保存します。
　外部に送信されることはありません。

2. 情報の利用目的
収集した情報は以下の目的で使用します。

・オンラインランキングの表示  
・ゲーム機能の提供  
・アプリの品質向上  
・広告配信（AdMob）

3. 第三者提供について
本アプリは、以下の場合を除き、ユーザー情報を第三者に提供しません。

・ユーザーの同意がある場合  
・法令に基づく場合  
・Google AdMob や Firebase など、アプリ運営に必要なサービス提供者に限る場合

4. データの管理
本アプリは Firebase（Google LLC）を利用してデータを安全に管理します。
Firebase のセキュリティルールに基づき、適切に保護されます。

5. 位置情報について
本アプリは GPS などの位置情報を取得しません。
